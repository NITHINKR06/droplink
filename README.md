# DropLink ⬡

**Instant bidirectional file transfer between PC and Mobile.**  
Drag, drop, done. Works on LAN and on the internet when hosted.

---

## Quick Local Run

```bash
# Terminal 1 — Backend
cd backend
pip install -r requirements.txt
uvicorn main:app --host 0.0.0.0 --port 8000 --reload

# Terminal 2 — Frontend (dev mode)
cd frontend
npm install
npm start
```

Open `http://localhost:3000` on PC.  
Open `http://YOUR_PC_IP:3000` on mobile (same WiFi).

## Deploy to Railway (Recommended — Free)

### Step 1: Build the frontend first
```bash
chmod +x build.sh
./build.sh
```
This compiles React and puts the output in `backend/static/`.  
After this, `backend/` is a self-contained deployable folder.

### Step 2: Push to GitHub
```bash
git init
git add .
git commit -m "initial commit"
git remote add origin https://github.com/YOUR_USERNAME/droplink.git
git push -u origin main
```

### Step 3: Deploy on Railway
1. Go to railway.app → New Project → Deploy from GitHub repo
2. Select your repo
3. Click Settings → set Root Directory to `backend`
4. Railway auto-detects railway.json and deploys
5. Go to Settings → Domains → generate a public URL

Your app is live at https://droplink-xxxx.railway.app

---

## Deploy to Render (Alternative — Free)

1. Push to GitHub (same as above)
2. Go to render.com → New Web Service
3. Connect your repo — Render auto-reads render.yaml
4. Hit Deploy

render.yaml includes a persistent disk so files survive restarts.
